# **App Name**: GradeWise

## Core Features:

- Dynamic Input Forms: Input fields for grades and weights; users can dynamically add or remove entries.
- Real-time Calculation: Real-time calculation of the final grade based on the inputted grades and their respective weights.
- Grade Visualization: Display the calculated final grade and individual grade components.
- Target Grade Calculator: Calculate the grade needed on a future assignment to achieve a desired overall grade, giving the user a planning tool.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a calm and focused learning environment.
- Background color: Light cyan (#E0FFFF) to create a soft and unobtrusive backdrop.
- Accent color: Soft yellow (#F0E68C) to highlight key information and call to action buttons.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, providing a modern, readable style.
- Use clean, minimalist icons to represent different grade components and actions.
- A clear, linear layout is critical for intuitive grade entry and review.
